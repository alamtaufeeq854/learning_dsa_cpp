A collection of C++ implementations covering Data Structures and Algorithms. This repository documents my learning journey and hands-on practice while following Abdul Bari's **Mastering Data Structures & Algorithms using C and C++** course on Udemy.

## Topics Covered

- C++ Fundamentals
- Recursion
- Arrays & Array ADT
- Strings
- Matrices
- Sparse Matrix & Polynomial
- Linked List
- Stack & Queue
- Trees, BST & AVL Trees
- Heap
- Sorting Algorithms
- Hashing
- Graphs
- Divide & Conquer
- Greedy Algorithms
- Dynamic Programming
- Backtracking
